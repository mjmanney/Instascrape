Instascrape

Developers can use this technique to get media from Instagram without using their API.

This workaround was found by Tom Dickinson who explains the technique and provides a link to his Python repository. My demo builds on this, although less complex, using JS only.

Search a hashtag and click on a photo to view an image. Note that the images displayed may have been posted in recent minutes.
